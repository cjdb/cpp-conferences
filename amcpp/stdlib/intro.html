Applied Modern C++

Course description

  • A two-day course
  • Details iconic parts of the Standard Library
  • We will be using C++17 and the Ranges TS
  • Most code will be C++11-compatible or portable

Course aims

This course aims to

  • introduce the salient features of the C++ Standard Library
  • demonstrate why the Standard Library is preferred
  • introduce the Ranges TS
  • demonstrate why adopting the Ranges TS right now is a great idea

Course structure (Day 1)

  1. Basics
  2. String manipulation
  3. Containers
  4. Input and output
  5. Iterators, sentinels, and ranges

Course structure (Day 2)

  1. Utilities
  2. Invocables
  3. Building a container
